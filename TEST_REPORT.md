### 2.4 路径覆盖示例

#### isThanks 函数
```javascript
function isThanks(question) {
  const thanksKeywords = ['谢谢', '感谢', 'thx', 'thanks', 'thank you'];
  return thanksKeywords.some(keyword => question.includes(keyword));
}
```
- **路径 1**: 包含感谢关键词 → 返回 true ✅
- **路径 2**: 不包含感谢关键词 → 返回 false ✅

#### getLatestPosts 函数
```javascript
async function getLatestPosts(limit = 10) {
  const safeLimit = Math.max(Math.min(parseInt(limit) || 10, 50), 0);
  const [latestPosts] = await pool.execute(
    `SELECT * FROM info_items WHERE is_private = 0 ORDER BY created_at DESC LIMIT ${safeLimit}`
  );
  return latestPosts;
}
```
- **路径 1**: 默认调用（无参数）→ limit=10 ✅
- **路径 2**: 正常数字 → limit=指定值 ✅
- **路径 3**: 超过最大值 → limit=50 ✅
- **路径 4**: 负数 → limit=0（SQL 可能报错）✅
- **路径 5**: 非数字 → limit=10 ✅

2. **安全测试**: 进行 SQL 注入、XSS 等安全漏洞测试
3. **兼容性测试**: 测试不同浏览器、不同设备的兼容性
4. **回归测试**: 每次代码变更后自动运行测试套件
